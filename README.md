# Time Series Forecasting Project

Репозиторий содержит проект по анализу и прогнозированию временных рядов на основе датасета Favorita.

## Структура репозитория

* `modules/` — пользовательские скрипты и вспомогательные функции для обработки данных.
* `saved_models/` — сохраненные веса моделей и дампы (pickle) для быстрого восстановления экспериментов.
* `Final.ipynb` — основной Jupyter Notebook: содержит EDA, обучение моделей и визуализацию результатов.
* `items.csv` / `oil.csv` / `stores.csv` — исходные данные (информация о товарах, котировки нефти и параметры магазинов).
* `requirements.txt` — список необходимых Python-зависимостей.
